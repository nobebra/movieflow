# 🎬 MovieFlow

**Личный кинодневник и книжный трекер** — простое веб-приложение для хранения оценок фильмов и книг.

🌐 [Открыть сайт](https://nobebra.github.io/movieflow/)

## ✨ Возможности

- 🎬 **Фильмы и 📚 Книги** — отдельные коллекции с переключением
- ⭐ **Оценки от 1 до 10** с цветовой индикацией:
- 📋 **Лист ожидания** — добавляй без оценки, оценивай позже
- ✏️ **Редактирование** — меняй название и жанры без удаления
- 🔍 **Поиск** по названию
- 🎯 **Фильтры и сортировка** по дате, рейтингу, жанрам
- 💾 **Экспорт в JSON** — сохраняй бэкап коллекции

## 🚀 Как использовать

1. Открой сайт по ссылке выше
2. Выбери тип: **Фильмы** или **Книги**
3. Заполни название, жанры и URL постера/обложки
4. Поставь оценку звёздами (или оставь пустым для листа ожидания)
5. Нажми **Добавить**

## 📦 Установка локально

Просто скачай файл `index.html` и открой в браузере
